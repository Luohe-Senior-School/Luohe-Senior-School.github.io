## 为啥会有这个网站？

1. 漯高的网站属实难看

2. 漯高的网站至今协议仍是是 `http`

3. 漯高的网站常年不更新

4. 漯高的网站中基本没有什么很有实用性的信息

因此，我们建立了这个崭新的网站
