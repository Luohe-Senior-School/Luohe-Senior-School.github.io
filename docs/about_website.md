# 关于本网站

## 为啥会有这个网站？

1. 漯高的网站属实简陋。

2. 漯高的网站至今协议仍是 `http` 这种不安全的协议。

3. 漯高的网站常年不更新，或是根本没有什么可以看的。

4. 漯高的网站中基本没有什么很有实用性的信息。

因此，我们建立了这个崭新的网站 ———— 漯高档案库，为让更多人更真切地了解我们的学校。
